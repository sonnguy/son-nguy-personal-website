# Son Nguy CV Build Package

Files:
- `resume-master-v2.md`: master resume content.
- `codex-cv-ui-prompt.md`: prompt for Codex to generate a 2-page ATS-friendly HTML CV.

Recommended workflow:
1. Put both files in a clean folder.
2. Open the folder with VS Code.
3. Ask Codex to follow `codex-cv-ui-prompt.md`.
4. Codex should create `son-nguy-cv.html`.
5. Open `son-nguy-cv.html` in Chrome.
6. Press Cmd + P / Ctrl + P.
7. Choose Save as PDF.
8. Use A4, disable headers/footers, try Scale 90–100%.
