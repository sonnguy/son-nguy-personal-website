# Chrome PDF Export

- Open `son-nguy-cv.html` in Google Chrome.
- Choose **Print** and select **Save as PDF**.
- Paper size: **A4**
- Pages per sheet: **1**
- Margins: **Default**
- Scale: **100%**
- Headers and footers: **Off**
- Background graphics: **On**

The document contains two explicit A4 print pages and uses embedded print CSS.
